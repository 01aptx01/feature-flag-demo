# Feature Flag Demo

This project demonstrates Feature Flags used in modern web development.

## Concept

Feature Flags allow developers to enable or disable features without deploying new code.

Example:

enable_new_ui = true

## Tech Stack

- React (Vite)
- Node.js (Express)
- Docker
- Docker Compose

## Run

docker compose up --build

Open:

http://localhost:5173

Admin panel:

http://localhost:5173/admin

## Demo

Toggle feature in admin panel to enable or disable the new UI.
